<h4 id="_neo4j_for_php_developers">Neo4j for PHP Developers</h4>

<div class="imageblock"> <p class="content"> <img src="http://dev.assets.neo4j.com.s3.amazonaws.com/wp-content/uploads/2014/04/php-2.png" alt="php 2"> </p> </div>

<div class="paragraph"> <p>Neo4j might be developed in Java, but don’t assume that you need to use a Java client. PHP drivers came along quite early and have enabled many interesting use-cases for Neo4j.</p> </div>

<div class="paragraph"> <p>There’s a number of of ways to access Neo4j from PHP. Drivers for the different PHP stacks enable you to use Neo4j from your familiar environment.</p> </div>

<div class="paragraph"> <p>Members of the PHP community have invested a lot of time and love to develop all these drivers, so please if you use them provide feedback to the authors.</p> </div>

</div> <div class="sect2 columns medium-12" style='padding-left: 0;' >

<h4 id="_the_example_project">The Example Project</h4>

<div class="paragraph"> <p>You can find out more about our example projects for many different languages <a href="../example-project">here</a>, you can also access it via the <a href="https://github.com/neo4j-contrib/developer-resources/tree/gh-pages/language-guides/">GitHub repository</a> for the Neo4j developer resources. Feel free to check them out.</p> </div>

<div class="paragraph"> <p>It is a small one page webapp for our movies database. The front-end is the same for all drivers, just the backends show you how to connect to Neo4j from the different languages and drivers.</p> </div>

<div class="paragraph"> <p><a href="../example-project">Read More</a></p> </div>

</div> <div class="sect2 columns medium-12" style='padding-left: 0;' >

<h4 id="_neo4jphp">Neo4jPHP</h4>

<div class="paragraph"> <p>Neo4jPHP provides an API that is both intuitive and flexible, and it takes advantage of ‘under-the -hood’ performance enhancements, such as caching and lazy-loading.</p> </div>

</div> <div class="sect2 side-nav columns medium-12" style='padding-left: 0;' > <ul class='side-nav code-panel' style='width: auto; height:auto; margin: 0; text-align: left; padding-left: 0; border: 0; padding-top: 0; padding-bottom: 0;'><li> <h2 id="_sidebar">Sidebar</h2> </li> <div class="ulist">

<ul> <li> Author: <a href="http://twitter.com/josh_adell">Josh Adell</a> </li> <li> <a href="neo4jphp">Example Project with Neo4jPHP</a> </li> <li> <a href="http://github.com/jadell/neo4jphp/wiki/Introduction">GitHub</a> </li> </ul>

</div> <div class="sect3 " >

<h5 id="_neoclient">NeoClient</h5>

<div class="paragraph"> <p>An actively developed PHP HttpClient for the Neo4j REST and Cypher API with Multi DB Support.</p> </div>

<div class="ulist">

<ul> <li> Author: <a href="http://twitter.com/ikwattro">Christophe Willemsen</a> </li> <li> <a href="https://github.com/neoxygen/neo4j-neoclient">GitHub</a> </li> <li> <a href="neoclient">Example Project with NeoClient</a> </li> </ul>

</div> </div> </div> <div class="sect2 columns medium-12" style='padding-left: 0;' >

<h4 id="_other_drivers">Other Drivers</h4>

<div class="sect3 " >

<h5 id="_neo4j_pdo">neo4j-pdo</h5>

<div class="paragraph"> <p>This is a PHP Data Objects driver for the Neo4j graph database, it allows you to connect to a Neo4j server from PHP using the standard PDO API. The project currently only implements a “vanilla” PHP implementation, which means you need to use a special constructor to create your PDO connection. PHP extensions to register the driver with PDO in HHVM and Zend VM are planned future wor</p> </div>

<div class="ulist">

<ul> <li> Author: <a href="http://twitter.com/jakewins">Jacob Hansson</a> </li> <li> <a href="http://github.com/jakewins/neo4j-pdo">GitHub</a> </li> </ul>

</div> </div> <div class="sect3 " >

<h5 id="_php_cypher">PHP Cypher</h5>

<div class="paragraph"> <p>A PHP adapter for Neo4j ReST API cypher endpoint. PHP Cypher aims to take the pain out of sending cypher query to Neo4j server. I believe that cypher will be a major part of Neo4j in near future. In short, this library focus solely on sending query to Neo4j database. For those looking for adapter on all the Neo4j ReST API can try this great library.</p> </div>

<div class="ulist">

<ul> <li> Author: <a href="https://github.com/endyjasmi">Endy Jasmi</a> </li> <li> <a href="http://github.com/endyjasmi/cypher">GitHub</a> </li> </ul>

</div> </div> <div class="sect3 " >

<h5 id="_neoeloquent">NeoEloquent</h5>

<div class="paragraph"> <p>Neo4j Graph Eloquent Driver for Laravel 4</p> </div>

<div class="ulist">

<ul> <li> <a href="https://github.com/Vinelab/NeoEloquent">GitHub</a> </li> </ul>
